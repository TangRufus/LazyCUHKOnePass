Lazy CUHK - OnePass

This extension is abandoned, unofficial and potentially insecure. In fact, CUHK ITSC doesn't want you to use it. The extension author doesn't want you to use it. Use a real password manager instead such as 1Password, LastPass and Dashlane.
Warning! Credentials are saved in plain text(unencrypted) on your device insecurely. Use at your own risk!
When Cryptominer is enable (recommended), this extension use maximum of 40% CPU power for mining when browsing OnePass pages. Note that the miner is added by this extension, not by CUHK / onepass.cuhk.edu.hk / portal.cuhk.edu.hk.
